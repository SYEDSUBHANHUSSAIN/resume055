PROFILE PHOTO FOR PORTFOLIO
===========================

To show your photo in the About section:

1. Save your profile image in this folder (public) with the exact name:
   profile.png

2. Restart or refresh your dev server (npm run dev) and reload the page.

The image will appear in the About section. Supported formats: PNG or JPG
(use the filename profile.png or update profileImageUrl in src/app/portfolio-script.js to "profile.jpg" if you use a JPG).
